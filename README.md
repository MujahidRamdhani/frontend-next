
## Fitur
- **Next.js**.
- **TypeScript**.
- **Joi**.
- **ShadCN**.
- **Zustand**.

## Instalasi

```bash
# Clone repositori
git clone repo
cd repo

# Instal dependensi
npm install
```

## Menjalankan Aplikasi
```bash
npm run dev
```
Aplikasi akan berjalan di `http://localhost:3000`.
